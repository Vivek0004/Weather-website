## Setup

  ##### Clone the repository
```bash
git clone https://github.com/Vivek0004/Weather-website.git
```
  ##### Move to the desired folder
```bash
cd \Weather-website
```
  ##### To install the dependencies, simply write
```bash
npm install
```

  ##### To run the app, simply write
```bash
npm start
```
